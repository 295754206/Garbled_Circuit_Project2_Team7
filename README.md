#### Input Format
The input begin with a binary string, which indicates the garbled input.

#### Output Format
Output lines, each line has a wire name and a number either 0 or 1.


#### Technical Specification

#### Sample Input:
> 11  
> W1  
> AND1 AND A B W1  
>  

#### Sample Output:
> W1 1
